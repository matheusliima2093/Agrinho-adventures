# Contributing to Agrinho Adventures

Obrigado por seu interesse em contribuir para o Agrinho Adventures! 

## Como Contribuir

### Reportar Bugs
Se você encontrar um bug, por favor crie uma issue com:
- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs comportamento atual
- Screenshots se possível

### Sugerir Melhorias
Tem uma ideia para melhorar o jogo? Abra uma issue com:
- Descrição da melhoria
- Justificativa
- Exemplos se possível

### Submeter Pull Requests
1. Faça um fork do repositório
2. Crie uma branch para sua feature (`git checkout -b feature/AmazingFeature`)
3. Faça commit das suas mudanças (`git commit -m 'Add some AmazingFeature'`)
4. Push para a branch (`git push origin feature/AmazingFeature`)
5. Abra um Pull Request

## Diretrizes de Código

- Use nomes descritivos para variáveis e funções
- Adicione comentários em código complexo
- Mantenha a formatação consistente
- Teste suas mudanças localmente

## Código de Conduta

Todos os contribuidores devem ser respeitosos e considerados com os outros.
