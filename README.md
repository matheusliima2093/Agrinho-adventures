# 🌾 Agrinho Field Quest

Um jogo de plataforma educativo sobre agricultura sustentável!

## 📖 Descrição

**Agrinho Field Quest** é um jogo estilo Super Mario onde você é um fazendeiro que deve:

- 🌱 Pular em plataformas
- 🌱 Coletar sementes boas e água
- 🦗 Derrotar gafanhotos, moscas e poluição
- 🏆 Chegar ao final de cada fase
- 👹 Enfrentar chefes nas últimas fases

## 🕹️ Como Jogar

### Controles
| Ação | Tecla |
|------|-------|
| Mover Esquerda | ← ou A |
| Mover Direita | → ou D |
| Pular | Espaço |
| Arremessar | ↑ ou W |

### Objetivo
1. Pular nas plataformas
2. Coletar sementes 🌱 e água 💧
3. Derrotar inimigos pulsando neles ou atirando
4. Chegar ao símbolo ✓ verde para completar a fase
5. Enfrentar chefes nas fases 3-5

## 🎮 Mecânicas

### 🌾 Itens
- **🌱 Sementes**: Colete para atirar em inimigos
- **💧 Água**: Colete para pulverizar inimigos (mais dano)
- **❤️ Vidas**: Comece com 3 vidas

### 👾 Inimigos
- **🦗 Gafanhoto**: Pule nele para derrota-lo
- **🪰 Mosca**: Esquive ou atire
- **💨 Poluição**: Inimigo aéreo, evite!

### 🏆 Fases
- **Fases 1-2**: Desafio Normal
- **Fases 3-5**: Incluem Chefes Finais

## 🚀 Como Executar

### Online (Recomendado)
1. Acesse: https://github.com/matheusliima2093/Agrinho-adventures
2. Clique em **Code** → **Codespaces** → **Create codespace on main**
3. Aguarde carregar
4. Clique com botão direito em `index.html` → **Open with Live Server**
5. 🎮 Jogo abre!

### Localmente
1. Clone: `git clone https://github.com/matheusliima2093/Agrinho-adventures.git`
2. Abra `index.html` no navegador
3. 🎮 Pronto!
